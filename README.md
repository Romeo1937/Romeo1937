### 2. High-Performance Android Termux Security Lab
* **The Software Engineering Challenge:** Building a fully functional, localized penetration testing lab within the severe storage, permission, and execution constraints of a non-rooted mobile operating system.
* **The Cybersecurity Challenge:** Porting, compiling, and configuring diverse offensive security toolchains (Python, Ruby, C-based dependencies) to execute flawlessly under a single unified environment.
* **The Hybrid Solution:** Designed and deployed a centralized security lab directly inside Termux on Android 14. ### 3. Android 14 Phantom Process Restriction Bypass
* **The Software Engineering Challenge:** Android 14 introduces aggressive native resource management (`phantom processes`), which automatically kills background terminal operations, severing persistent Node.js scripts, automation frameworks, and long-running security scans.

Safely deactivated native background process limits and modified execution policies without compromising primary device stability or risking host OS integrity.
